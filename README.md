# Cryptography Application

This project is a C++ cryptography application that allows users to securely encrypt and decrypt files, generate hashes, and create or verify digital signatures. It uses established cryptographic libraries such as OpenSSL and Crypto++ to implement algorithms including AES, RSA, Blowfish, MD5, and SHA-256. The application is designed with a modular structure separating the user interface from the cryptographic engine, making it easier to maintain, extend, and integrate into other projects.
